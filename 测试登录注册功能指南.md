# 测试登录/注册功能指南

## 环境配置要求

在测试登录/注册功能之前，需要确保以下环境已正确安装：

1. **JDK 17**：后端项目需要JDK 17运行环境
2. **Maven**：用于构建和运行后端项目
3. **Node.js**：已安装（v22.20.0）
4. **npm**：已安装（10.9.3）

## 配置步骤

### 1. 安装JDK 17

- 下载JDK 17：从Oracle官网或OpenJDK下载
- 安装后配置环境变量：
  - 设置`JAVA_HOME`指向JDK安装目录
  - 将`%JAVA_HOME%\bin`添加到`PATH`环境变量

### 2. 安装Maven

- 下载Maven：从Apache Maven官网下载
- 安装后配置环境变量：
  - 设置`MAVEN_HOME`指向Maven安装目录
  - 将`%MAVEN_HOME%\bin`添加到`PATH`环境变量

### 3. 启动后端服务

1. 打开命令行，进入后端项目目录：
   ```
   cd d:\02_Academic\iFM\backend
   ```

2. 使用Maven构建并运行项目：
   ```
   mvn spring-boot:run
   ```

   等待服务启动完成，通常会显示类似以下信息：
   ```
   Started IfmServiceApplication in X.XXX seconds
   ```

### 4. 启动前端服务

1. 打开另一个命令行，进入前端项目目录：
   ```
   cd d:\02_Academic\iFM\Frontend
   ```

2. 启动前端开发服务器（如果有）：
   ```
   npm install
   npm run dev
   ```

   或者直接在浏览器中打开`login.html`文件：
   ```
   d:\02_Academic\iFM\Frontend\html\login.html
   ```

## 测试步骤

### 1. 注册测试

1. 打开登录/注册页面
2. 点击"注册"选项
3. 填写以下信息：
   - 用户名：任意3-10个字符
   - 手机号：11位真实格式手机号（如13800138000）
   - 密码：至少6位
   - 确认密码：与密码相同
4. 点击"注册"按钮
5. 预期结果：
   - 注册成功提示
   - 自动切换到登录界面

### 2. 登录测试

1. 在登录界面填写：
   - 手机号：刚才注册的手机号
   - 密码：刚才设置的密码
2. 点击"登录"按钮
3. 预期结果：
   - 登录成功提示
   - 跳转到上传页面

## 常见问题及解决方法

### 1. 后端服务启动失败

- 检查JDK和Maven版本是否正确
- 检查端口8080是否被占用
- 查看控制台错误信息

### 2. 前端无法连接后端

- 确保后端服务已启动
- 检查API地址是否正确（应为http://localhost:8080/api/user/...）
- 检查浏览器控制台是否有跨域错误

### 3. 注册失败

- 检查手机号格式是否正确（11位数字）
- 检查密码长度是否至少6位
- 检查用户名长度是否在3-10位之间

### 4. 登录失败

- 检查手机号是否正确
- 检查密码是否正确
- 确保用户已注册
